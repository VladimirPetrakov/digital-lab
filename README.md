# Обзор решения
## Описание
Алгоритм конвертации заключается в построчном обходе матрицы.
Обход происходит без учета строк и столбцов, в которых заведомо не может содержаться шаблон.

## Классы
- ***Matrix*** - класс-обертка над двумерным массивом, в котором реализованы методы заполнения и копирования.

- ***MatrixConverter*** - класс, конвертирующий матрицу по шаблонной матрице. Содержит правила конвертации.
